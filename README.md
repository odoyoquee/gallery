## SETUP REQUIREMENTS
- Python 3.6
- Internet Connection
- Run the following commands on your terminal:
`$ git clone https://github.com/odoyoquee/gallery`
- Navigate to the folder
 `$ cd <folder/file name>`
- To run the application;
 `$ python3.6 manage.py runserver`
